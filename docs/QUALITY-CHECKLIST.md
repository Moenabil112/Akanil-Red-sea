# Human Quality Checklist

Before accepting Claude Code's first release, verify:

## Identity
- [ ] Akanil is clearly the founder and executive operator.
- [ ] The Gateway is permanent.
- [ ] The Forum is the first activation programme.
- [ ] The future chamber is not presented as existing.

## Morocco
- [ ] Morocco's industrial, financial, standards, technology and export
      value is prominent.
- [ ] The page feels relevant to Moroccan institutions and companies.
- [ ] French and Arabic copy is institutionally credible.

## Sudan
- [ ] Sudan is presented as a value-chain partner.
- [ ] The language avoids extractive framing.
- [ ] Shared value and local capability are visible.

## Visual quality
- [ ] The site does not look like a generic AI landing page.
- [ ] Cards, glows and generated imagery are restrained.
- [ ] Layout rhythm changes meaningfully between sections.
- [ ] Generated image text is not used as final interface copy.
- [ ] Concept art is labelled where required.

## Interaction
- [ ] Motion explains route, process, transformation or trust.
- [ ] No scroll hijacking.
- [ ] Keyboard and touch interaction work.
- [ ] Reduced-motion behavior is visible.

## Language
- [ ] Arabic is native RTL.
- [ ] French and English are independently edited.
- [ ] Language switching preserves context.
- [ ] Mixed-direction content renders correctly.

## Trust
- [ ] No target organization is presented as a confirmed partner.
- [ ] No fake metrics, awards, reviews or endorsements appear.
- [ ] Conceptual routes are not presented as operational.
- [ ] AI is not presented as a final decision-maker.

## Engineering
- [ ] Build, lint and type checks pass.
- [ ] No production secrets are present.
- [ ] No approved archive is shipped in the public bundle.
- [ ] Large source PNG assets are not loaded on first view.
- [ ] No deployment occurred without authorization.
