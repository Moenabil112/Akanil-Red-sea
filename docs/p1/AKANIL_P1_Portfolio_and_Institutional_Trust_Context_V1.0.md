# 10. VALURA — Initial Public Project Context

## 10.1 Proposed Public Name

**VALURA — Value-Added Agro-Industrial Complex**  
**فالورا — مجمع الصناعات التحويلية الزراعية**

## 10.2 Project Type

Integrated agricultural processing and value-addition complex.

## 10.3 Core Value

VALURA is designed to transform Sudanese agricultural raw materials into higher-value processed, packaged, and exportable products.

The project addresses the gap between:

- exporting agricultural crops as raw commodities;
- creating value through processing, extraction, sorting, sterilization, packaging, storage, and industrial by-products.

## 10.4 Main Components

The preliminary blueprint describes six integrated industrial units:

- agricultural oil processing for sesame, groundnuts, and watermelon seeds;
- medicinal and wild-herb oil extraction;
- optical and AI-assisted sorting;
- automated packaging lines;
- vegetable and fruit sterilization;
- solar-powered cold storage.

The design also includes:

- hybrid energy;
- waste-reduction and by-product reuse;
- agricultural supply integration with RWAFID.

## 10.5 Indicative Figures

The following are **preliminary blueprint estimates**, not achieved results:

- Indicative capital requirement: **USD 8–12 million**
- Detailed preliminary CAPEX range in the source model: approximately **USD 8.8–12.5 million**
- Indicative capacity: **50–80 tonnes/day**
- Indicative land requirement: **50–80 feddans**
- Direct employment target: **250+**
- Indirect employment target: **500+**
- Indicative payback period: **3–4 years**
- Indicative IRR shown in the source: **28–35%**

### Publication Rule

The public card may show only a limited selection of these figures, clearly labelled:

> Preliminary estimates from VALURA Blueprint V1.0 — March 2026.

Do not present IRR, revenue, or payback as guaranteed.

## 10.6 Current Stage

Recommended public wording:

The source document identifies possible locations in:

- Kassala;
- Gedaref;

- access is guaranteed;
