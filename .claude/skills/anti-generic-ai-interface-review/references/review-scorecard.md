# Anti-Generic Interface Scorecard

Score each item from 1 to 5.

| Dimension | 1 | 3 | 5 |
|---|---|---|---|
| Institutional authorship | Generic template | Some project specificity | Could only belong to Akanil |
| Moroccan relevance | Not visible | Mentioned | Drives narrative and actions |
| Visual hierarchy | Flat or chaotic | Understandable | Deliberate editorial rhythm |
| Interaction meaning | Decorative | Partly useful | Clarifies process and decisions |
| Image credibility | Misleading | Mixed | Status-aware and purposeful |
| Arabic RTL | Broken/mirrored | Functional | Fully authored |
| Trust | Claims and logos | Basic disclaimers | Evidence, status and process visible |
| Performance | Heavy by default | Acceptable | Motion and media are disciplined |
| Accessibility | Major barriers | Basic support | Integrated into design |
| Content clarity | Slogans | General explanation | Precise stakeholder proposition |

Suggested result:
- 43–50: Pass.
- 34–42: Conditional Pass.
- Below 34: Fail and redesign.
A critical trust or accessibility failure can override the numeric score.
