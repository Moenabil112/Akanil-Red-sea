# Anti-generic AI interface review

Review the running interface, not only screenshots or source files.

## Immediate rejection signals

Flag as high priority when the interface contains:

- a generic gradient hero with interchangeable copy;
- many identical rounded cards;
- decorative glass panels without an information function;
- excessive glow, particles or floating orbs;
- random 3D objects;
- fake dashboards, metrics or charts;
- AI-generated imagery in every section;
- slogans without process, evidence or stakeholder value;
- repeated “learn more” buttons;
- unverified logos;
- an event-site identity replacing Akanil and the Gateway;
- weak Arabic typography or mirrored LTR layout;
- motion that delays access to content.

## Review dimensions

Score each dimension from 1 to 5:

1. institutional authorship;
2. Akanil identity;
3. Moroccan stakeholder relevance;
4. narrative clarity;
5. visual hierarchy;
6. interaction meaning;
7. image credibility;
8. Arabic RTL quality;
9. accessibility;
10. performance discipline.

Read `references/review-scorecard.md`.

## Required output

Return:

- overall score;
- five strongest elements;
- five credibility risks;
- sections that look template-generated;
- concrete redesign actions;
- elements to remove, not merely improve;
- pass, conditional pass or fail.

## Pass rule

A page does not pass merely because it is polished.

It passes when a Moroccan institutional stakeholder can identify:

- the owner;
- the purpose;
- Morocco's value;
- Sudan's role;
- what is confirmed;
- the next institutional action;

without being distracted by decorative technology.
